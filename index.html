simply_clean.css

A simple css library to make most elements look decent on static pages. It is not intended for any large projects.

Built to be

  • Portable
  • Responsive
  • Classless
  • Customizeable

Installation

Install with npm install simply_clean.css and include it in the head of your page with node_modules/simply_clean.css/style.css.
